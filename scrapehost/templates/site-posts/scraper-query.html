Scraper Queries

Queries are written in python, and can look something like this:

images = document.find_all('img')


for img in images:
    src = img.get('src')

    if src:
        if 'http' not in src:
            src = urlparse.urljoin(url, src)

        self.data.append(src)


This query will look for images and store them in the scrapers own database.
Storing data in the database is done here: 

self.data.append(src)
